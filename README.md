# 🔐 Password Manager (Python Project)

A simple desktop Password Manager built using Python.  
It stores and encrypts user passwords securely.

---

## 🚀 Features
- Add website credentials (username & password)
- Encrypt passwords using Fernet encryption
- Store data in SQLite database
- Simple GUI using Tkinter

---

## 🛠️ Tech Stack
- Python
- Tkinter (GUI)
- SQLite (Database)
- Cryptography (Encryption)

---

## ▶️ How to Run

```bash
pip install cryptography
python app.py
