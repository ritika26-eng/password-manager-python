# 🔐 Password Manager (Python)

A simple desktop Password Manager built using Python, Tkinter, SQLite, and encryption.

---

## 🚀 Features

- Add and save website credentials
- Encrypt passwords using Fernet encryption
- Store data securely in SQLite database
- Simple GUI using Tkinter

---

## 🛠️ Tech Stack

- Python
- Tkinter (GUI)
- SQLite (Database)
- Cryptography (Fernet Encryption)

---

## ▶️ How to Run

```bash
pip install cryptography
python app.py
